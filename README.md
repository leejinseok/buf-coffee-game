## 게임에 참가하시겠습니까? 커피게임?
### 설명
네이버 제비뽑기에 물려서 만든 모임

### 게임의 규칙
폴더명 : {name}/g{n} (ljs/g0, ljs/g1)
파일명 : ljs/g1/index.html, ljs/g1/index.js
파일형태 : 화면형, 콘솔형 (cli)

### 개발환경
- html5
- node 18.12.0

### 브랜치명
- feature/{name}/사다리게임 -> main 브랜치로 pr
